### Requirements

#### For compiling the Newer game hacks:

- OS and setup where you can compile LLVM and Clang
- devkitPPC
- Python 2.x
- Python libraries: PyYAML, pyelftools

#### For using the Newer tools:

- Windows, Linux or Mac OS
- Python 2.x
- PyQt
- Possibly something else I forgot?
